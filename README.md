# zeroOnSides

Write a PHP program that pushes half of the zeros in a given array to the beginning of the array, and the other half to the end.

Examples 1:
Input: [3,5,6,0,7,0,1]
Output: [0,3,5,6,7,1,0]

Examples 2:
Input: [5,0,0,6,0,8]
Output: [0,0,5,6,8,0]

Examples 3:
Input: [1,2,3,0,0,0,0]
Output: [0,0,1,2,3,0,0]

Examples 4:
Input: [1,2,3]
Output: [1,2,3]
